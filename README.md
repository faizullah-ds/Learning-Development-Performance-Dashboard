# Learning & Development Performance Dashboard

The **Learning & Development Performance Dashboard** is an interactive HR analytics solution designed to monitor, analyze, and improve employee training effectiveness across the organization. Built using **Power BI** with advanced **DAX measures**, the dashboard provides end-to-end visibility into training sessions, trainer and trainee performance, adherence levels, and departmental training trends.
